Escreva uma hierarquia de herança para as classes Quadrilátero, Trapézio, Paralelogramo, Retângulo e Quadrado.

Faça a hierarquia o mais profunda (isto é, com muitos níveis) possível. 

Especifique as variáveis de instância e os métodos para cada classe. 

As variáveis de instância private de Quadrilátero devem ser os pares de coordenadas x−yx−y para os quatro pontos que delimitam o Quadrilátero. 

Escreva um programa que instancia objetos de suas classes e gera saída da área de cada objeto (exceto Quadrilátero).

public class Quadrilatero

{

   private int x;

   private int y;

   public void area(){

      System.out.println("+x*y+");

    }

}

public class Retangulo extends Quadrilatero

{

   private int x;

   private int y;

    public void area(){
      super.area();
    }

}

public class Quadrado extends Quadrilatero

{

   private int x;

   private int y;

public void area(){
      super.area();
    }

}

Public class Trapézio extends Quadrilatero

{

    public void area(){
      super.area();
    }

}

public class Paralelograma

{

    }
